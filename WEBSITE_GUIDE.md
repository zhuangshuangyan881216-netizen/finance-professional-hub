# 财务专业中心 - 网站使用指南

## 项目概述

**财务专业中心** 是一个综合性的财务专业平台，融合了个人博客、资源导航站和职业作品集三大功能模块。该网站为财务专业人士提供深度的知识分享、精选的工具资源和专业能力展示。

## 设计理念

### 色彩系统
- **蓝色 (#1E40AF)** - 主色调，代表专业与信任
- **黄色 (#F59E0B)** - 活力色，强调数据亮点和行动
- **绿色 (#10B981)** - 增长色，表示正向趋势
- **橙色 (#F97316)** - 热情色，驱动学习与创新

### 排版
- **标题字体** - Playfair Display（优雅衬线体）
- **正文字体** - Inter（现代无衬线体）

## 网站结构

### 1. 首页 (/)
**功能**：网站入口和内容导航
- 英雄区 - 品牌展示和核心价值主张
- 最新文章 - 博客文章预览
- 资源导航 - 工具和资源快速访问
- 职业作品集 - 项目成果展示
- 订阅区 - 邮件订阅功能

### 2. 博客页面 (/blog)
**功能**：文章发布和知识分享
- **搜索功能** - 快速查找相关文章
- **分类过滤** - 按主题分类（财务分析、Excel、税务、Power BI、Power Query）
- **文章卡片** - 展示标题、摘要、作者、日期、阅读时间
- **订阅功能** - 获取最新文章通知

**主要分类**：
- 财务分析 - 财务报表分析、经营分析、决策支持
- Excel 教程 - 函数、数据透视表、VBA 自动化
- 税务规划 - 税收优化、合规管理、风险防控
- Power BI - 仪表板设计、DAX 函数、数据可视化
- Power Query - 数据导入、转换、清洗

### 3. 资源导航页面 (/resources)
**功能**：工具、模板和学习资源集合
- **Excel 工具库** - 财务模板、分析工具、VBA 脚本
- **Power BI 资源** - 仪表板模板、DAX 函数库、最佳实践
- **Power Query 教程** - 数据导入指南、清洗方法、M 语言参考
- **财务模型库** - 预测模型、估值工具、分析模板
- **推荐链接** - 官方文档和学习平台

**资源特性**：
- 下载计数 - 显示资源热度
- 用户评分 - 5 星评级系统
- 分类标签 - 快速识别资源类型

### 4. 作品集页面 (/portfolio)
**功能**：展示专业能力和项目成果
- **项目展示** - 6+ 个代表性项目
- **过滤功能** - 按技术栈分类（Power BI、Excel、Python、财务模型）
- **项目指标** - 显示关键成果数据
- **案例研究** - 深度项目分析和成果展示
- **统计数据** - 已完成项目数、满意度、开源贡献

**项目类型**：
- Power BI 仪表板 - 数据可视化和分析系统
- Excel 模型 - 财务预测和优化工具
- Python 项目 - 机器学习和数据分析
- 财务模型 - 三表联动、估值、预测

## 技术架构

### 前端技术栈
- **框架** - React 19 + Wouter（客户端路由）
- **样式** - Tailwind CSS 4 + 自定义 CSS 变量
- **组件库** - shadcn/ui
- **图标** - Lucide React
- **动画** - Framer Motion（预留）

### 项目结构
```
client/
├── src/
│   ├── pages/          # 页面组件
│   │   ├── Home.tsx    # 首页
│   │   ├── Blog.tsx    # 博客页面
│   │   ├── Resources.tsx # 资源导航
│   │   └── Portfolio.tsx # 作品集
│   ├── components/     # 可复用组件
│   │   ├── Navigation.tsx # 导航栏
│   │   └── Footer.tsx     # 页脚
│   ├── App.tsx         # 路由配置
│   └── index.css       # 全局样式
└── public/             # 静态资源
```

## 自定义指南

### 修改色彩主题
编辑 `client/src/index.css` 中的 `:root` 变量：
```css
--color-primary-blue: #1E40AF;
--color-accent-yellow: #F59E0B;
--color-accent-green: #10B981;
--color-accent-orange: #F97316;
```

### 添加新博客文章
编辑 `client/src/pages/Blog.tsx` 中的 `blogPosts` 数组，添加新文章对象。

### 添加新资源
编辑 `client/src/pages/Resources.tsx` 中的 `resourceCategories` 数组。

### 添加新项目
编辑 `client/src/pages/Portfolio.tsx` 中的 `projects` 数组。

### 修改导航链接
编辑 `client/src/components/Navigation.tsx` 中的 `navItems` 数组。
